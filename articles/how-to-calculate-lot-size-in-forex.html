How to Calculate Lot Size in Forex

To calculate lot size in forex, first decide the dollar amount you are willing to risk, then divide that risk by the stop loss distance and pip value.

Educational note: This guide is for learning and risk planning only. It is not financial advice.

Lot size formula

A simple lot size formula is: risk amount divided by stop loss pips divided by pip value per standard lot. The exact result may vary by broker and instrument.

Example with a $1,000 account

If the account is $1,000 and risk is 1%, the risk amount is $10. If the stop loss is 20 pips and pip value is $10 per standard lot, the estimated lot size is 0.05 standard lots.

Common lot size mistakes

  • Choosing lot size based on desired profit instead of risk.
  • Increasing lot size after a losing trade.
  • Using the same lot size for every pair without checking pip value.
  • Ignoring spread, commission, and slippage.

A practical beginner rule

Beginners can start by choosing a small fixed risk percentage, using the calculator before each trade, and reviewing whether the planned loss feels emotionally manageable.

Frequently Asked Questions

How do I calculate lot size in forex?

Calculate your risk amount first, then divide it by stop loss pips and pip value per standard lot.

What lot size should I use for a $100 account?

There is no universal lot size. It depends on risk percentage, stop loss distance, pip value, and the broker's minimum trade size.

Why does stop loss distance affect lot size?

A wider stop loss needs a smaller lot size to keep the same dollar risk. A tighter stop loss allows a larger lot size but may be easier to hit.
