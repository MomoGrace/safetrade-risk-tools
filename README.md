# SafeTrade Risk Tools

A beginner-friendly **Forex, EA and Prop Firm Risk Calculator** website.

This project includes:

- Conservative EA Setup Calculator
- Prop Firm Risk Calculator
- Forex Drawdown Calculator
- SEO content section
- FAQ section
- Lead capture section placeholder
- Risk disclaimer
- Basic SEO metadata
- `robots.txt`, `sitemap.xml`, and `ads.txt` placeholders

> Important: This website is for educational and risk-planning purposes only. It does not provide financial advice, trading signals, or profit guarantees.

---

## Tech Stack

- Vite
- React
- Tailwind CSS

---

## Run Locally

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Upload to GitHub

1. Create a new GitHub repository, for example:

```text
safetrade-risk-tools
```

2. Upload all files from this folder.

Or use Git:

```bash
git init
git add .
git commit -m "Initial SafeTrade Risk Tools website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/safetrade-risk-tools.git
git push -u origin main
```

---

## Deploy to Vercel

1. Go to Vercel.
2. Click **Add New Project**.
3. Import your GitHub repository.
4. Keep the default Vite settings.
5. Click **Deploy**.

After deployment, replace all `https://example.com` references in:

- `public/robots.txt`
- `public/sitemap.xml`

with your real website domain.
